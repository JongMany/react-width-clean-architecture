# Object-oriented React Architecture
Object-oriented react architecture sample code.  
if you leave an issue or a pull request, we will reflect the insufficient part or improvement. ☺️  

## Use Stack
Typescript, Webpack, React, Recoil, Emotion

## Directory Structure
```
./src
├─ constants
├─ adapters
│  ├─ infrastructures
│  │  └─ interfaces
│  ├─ repositories
│  │  └─ interfaces
│  ├─ presenters
│  │  ├─ interfaces
│  │  └─ converter-interfaces
│  ├─ dto
│  └─ vm
├─ services
│  └─ converters

   └─ hooks
      └─ ...
```

## Run Project
### 1. Mock Server
#### Install
```shell
# ./mock-server
$ npm install
```
#### Start
```shell
# ./mock-server
$ npm start
```

### 2. Project
#### Install
```shell
# ./
$ npm install
```
#### Start
```shell
# ./
$ npm start
```
